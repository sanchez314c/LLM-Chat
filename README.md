## Quick Start

```bash
# For basic functionality (v0.1.0)
cd LLMChat-v0.1.0
pip install -r requirements.txt
export XAI_API_KEY=your_key
python voyeur_chat.py

# For full features (v2.0.0)
cd LLMChat-v2.0.0
pip install -r requirements_core.txt
export XAI_API_KEY=your_key
python main.py
```
